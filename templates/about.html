About CashView

About the App
What is CashView?

CashView is a financial dashboard that allows you to securely connect and view all of your financial accounts in one location! Have several banks that you have holdings with? CashView is a convenient solution to tracking them all in one place!

What else can I do with CashView?

CashView also enables users to set up BudgetTracker text notifications for eligible account types (checking and credit)!

What technologies were used to build CashView?
  • Front-end: JavaScript, Axios, HTML5, CSS, Bootstrap, jQuery, GoogleChart.js
  • Back-end: Python, Flask, ORM: SQLAlchemy, Jinja2, WTForms, Bcrypt, Cron
  • Database: Postgres
  • APIs Integrated: Plaid, Twilio
